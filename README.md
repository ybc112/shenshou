# 如意神兽发射台合约

第一版合约实现“创建神兽、发行代币、交易喂养、灵气进化、销毁分红奖励”的链上闭环。

## 合约模块

- `RuyiBeastLaunchpad`: 发射台工厂合约，负责创建神兽项目和部署专属 Token。
- `RuyiBeastToken`: 神兽 ERC20，负责买卖税、灵气累计、阶段进化、持币分红。
- `RuyiBeastVault`: 税池合约，负责进化池、福运池、风险池、供养池、金库、销毁和分红储备。

## 当前版本机制

- 买入税默认 `3%`: 进化池 `1.5%`，福运池 `0.5%`，风险池 `0.5%`，供养/奖励池 `0.5%`。
- 卖出税默认 `5%`: 进化池 `2%`，福运池 `1%`，风险池 `1%`，供养/奖励池 `0.5%`，平台金库 `0.5%`。
- 交易税以本项目 Token 形式进入 `Vault`，进化池累计为灵气值。
- 灵气值达到阈值后可触发进化。
- 进化时默认销毁进化池 `50%`，释放奖励池 `50%` 作为持币分红。
- 分红采用“累计每份收益”的可领取模型，用户主动调用领取，避免遍历持有人。

## 命令

```bash
npm install
npm run compile
npm test
npm run web
```

打开前端页面：

```text
http://127.0.0.1:5173/
```

前端页面位于 `web/`，当前包含首页、神兽大厅、创建神兽、进化榜、奖励中心、数据中心和帮助中心。

前端不再内置神兽素材图片，神兽展示使用 CSS 符文徽章和链上数据渲染。创建神兽时可选择本地图片，页面会在浏览器内压缩并生成 `data:application/json;base64,...` 元数据 URI 后传入合约，不上传 IPFS。

## 本地真实链上数据

开发联调推荐开三个终端：

```bash
npm run node
npm run deploy:local
npm run seed:local
npm run web
```

- `deploy:local` 会部署 `RuyiBeastLaunchpad` 和 `RuyiBeastVault`。
- 部署脚本会自动写入 `web/config.js`，前端会读取真实合约地址。
- `seed:local` 会创建几只神兽，并模拟真实交易税、灵气累计、进化销毁和分红池数据。
- 如果要连接钱包，请把钱包网络切到 `http://127.0.0.1:8545`，Chain ID 为 `31337`。
