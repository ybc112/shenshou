HELP CENTER

帮助中心

这里按玩家实际操作来说明创建、Mint、奖励和安全注意事项。

怎样创建神兽

填写神兽名称、Token 信息、供应量和进化阈值，添加神兽图片后确认钱包交易。需要 Mint 发射时，再填写 Mint 池数量、单价和截止时间。

怎样参加发射

进入神兽详情，看到“Mint中”后输入数量并确认支付。Mint 成功后，Token 会进入当前钱包，BNB 会和预留 Token 自动加入 Pancake 池子，LP 默认进黑洞。

怎样获得奖励

持有神兽 Token 后，交易税会进入税池。神兽进化时会销毁一部分并释放分红，也会开启灵符和本命号码奖励轮次。

项目方怎样运营

发射台管理员可在兽巢详情里配置奖励参数、手动开奖、DEX 路由、回购地址、自动回购比例和自动加池比例。

安全提醒

只认准页面显示的合约地址和钱包弹窗金额。创建、Mint、开奖和 DEX 设置都会弹出钱包确认，确认前请核对操作内容。
